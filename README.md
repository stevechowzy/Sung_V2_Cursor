## 写作指南
### 章节节奏控制
1. **日常章节**（2000-2500字）
   - 包含1个主指令+1个情感事件
2. **升级章节**（3000-3500字）
   - 系统升级+新能力解锁+角色关系突破
3. **转折章节**（4000+字）
   - 时空悖论爆发+多女主命运交汇
